一. 参加培训的守则

1. 若您在本课程培训协议上签字，或勾选网站上“我同意”按钮并按照金程教育在线注册程序成功注册为金程教育学员，即表示您在注册及签订协议之前认真阅读了本用户协议，同意并签署了本用户协议。本协议您签署后已构成在您和金程教育之间的具有法律约束力的合同，您必须遵守本协议以及金程教育的所有政策与规章。

2. 在您缴费完成后甲方即提供培训服务，仅在您有效的培训课程期间内有效。培训资格不得转让且不设退款，如果课程特殊情况可以申请延班或者转班，具体操作按照相关规定。

3. 您有责任向我们通报关于您的任何特定健康或其他需注意问题。如遇紧急情况，我们将会拨打您提供的紧急联系号码通知紧急联系人。

4. 签订课程培训协议后，您将获发上课凭证。如未携带有效的凭证，在您提供身份证原件的情况下，是否获准进入金程教育课堂将由我们酌情决定。

5. 请您避免携带任何贵重物件进入培训课堂，如已经带入培训课堂，则完全由您个人承担保管责任。如果您的衣服、财物或其他个人物品发生丢失或任何损坏，我们在任何情况下对该损失或损坏不负有任何责任，并且无需承担任何赔偿责任。

6. 学员必须妥善保管所注册的金程一号通会员用户名和密码，如果丢失，造成的损失将由本人承担全部后果，金程教育不对学员所发布信息的删除或储存失败负责。学员对利用该用户名及密码所进行的一切活动负全部责任，因此所衍生的任何损失或损害，金程教育不承担任何责任。

二. 培训双方的权利与义务

1. 甲方的权利与义务

1.1. 甲方有义务为乙方提供相应课程的讲授和配发对应课程的讲义（面授班）；

1.2. 甲方有义务在现有技术上维护整个网校教育平台的正常运行，并努力提升和改进技术， 使学员的网络教育活动得以顺利进行（网络班）。

1.3. 甲方取得关于您的相关报班信息，以及您紧急联系人的资料。将仅会按照甲方的隐私政策使用、披露或处理这些信息，在未获得乙方同意的情况下，不得擅自向第三方透露乙方信息。但甲方可能不时为课堂、培训活动以及相关作品进行记载、拍照或拍摄影片，用于培训跟踪、质量控制和营销用途，而您在上课及参与活动以及在这些上课和活动过程中制作的作品可能会被记载、拍照或摄入影片之内。您同意我们可以就培训、质量控制和营销活动使用该等记载文件、照片和影片。

1.4. 甲方对于学员在金程教育学习平台上或课堂中的不当行为，包括但不限于发布违法信息、诽谤金程的不良言论、欺侮行为、骚扰行为、毁坏金程教育任何财产或设备等，甲方有权随时作出删除相关信息或将培训学员带离课堂，并且取消其培训资格。如果您的培训资格基于不当行为被取消，您将无权要求返还培训费用。

1.5. 金程教育应本着诚实信用的原则向学员提供培训服务，不得随意中断或停止提供服务。

1.6. 自签订本协议起,甲方不接受乙方任何的退费申请。金程教育的销售代表、客服等任何级别、任何部门员工的口头、书面同意退费、减免培训价格的承诺均不能对抗本规定。

2. 乙方的权利与义务

2.1. 在取得培训资格前须事先支付所有培训费和学习资料费。在学籍期间内，可享受甲方提供的约定的课程、各类咨询、信息查询等相关服务；

2.2. 乙方需向甲方提供详尽、准确的个人资料并及时更新，以保障培训服务的顺利开展。若学员提供任何错误、不实、过时或不完整的资料导致培训服务的延迟或无法开展，造成损失学员自行承担，同时甲方将保留结束或终止其培训资格的权利。

2.4. 甲方根据实际情况给予乙方一定的优惠或协助，乙方有义务在任何情况下，都不得将相应优惠或协助的内容透露给第三方。如甲方发现乙方有此类行为发生，则甲方有权收回之前给予乙方的优惠。收回方式包括但不仅限于停止提供培训服务。

三. 插班、延班、转班、重读的规定

1. 插班：学员因各种原因错过报名时间的，可插班学习。插班学员应缴纳该课程的全部费用及资料费等相关费用，金程教育提供所缺课程的网络学习课程；

2. 转班：开课1个月后概不办理此手续。学员所转入新班级学费高于原所报班级剩余学费（原所报班级的学费减去已上课时学费，已上课学费按学时比例计算），应补足费用差价（差价等于转入班级学费减去原所报班级剩余学费）；学员所转入新班级学费低于原所报班级剩余学费的，转班后学费差价不做退费；

3. 延班：学员因实际情况，需要延期就读所报班级的，应于开课日起1个月之内到学校办理延班手续，开课1月后概不办理此手续。学员延班后，需退回原听课证，并关闭相应权限。延班自办理日起365天内有效，过期所延课程自动失效，且不做退费处理。延班仅限一次，不可连续延班且延班后不可退班。

4. 重读：需申请重读的学员，需按照以下政策申请重读（仅限CFA&FRM）：

4.1. 面授课程出勤率或网络课程学习率需达到85%（面授课程必须签到，出勤情况以签到表为准）方可申请重读；

4.2. 学员需在考试成绩公布后（官方公布的成绩查询日开始）30天内凭考试未通过的成绩单及下期考试报名确认信，办理重读申请，二者缺一不可，重读费用为所读班级原价费用的15%；已报考而未参加当期考试或未报考当期考试的学员，凭下一期考试报名确认信申请重读，重读费用为所读班级原价费用的50%；

4.3. 重读仅限一次，且重读不可延班、转班、退班。

4.4. 百题班、IPS写作班、押题班等少于全景班课时量的班级，不享受重读政策；
